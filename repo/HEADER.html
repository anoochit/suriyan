Suriyan is delivered distribution from Ubuntu, developed by ThaiOpenSource.org and local community in Thailand. Suriyan is a complete and ready desktop operating system. You can use it as Live CD or install to your machine. Suriyan is supported by Software Industry Promotion Agency (Public Organization) a.k.a SIPA.

To use this repository install

  • Ubuntu 12.04 LTS Precise Pagolin suriyan-repository_1.1-5_all.deb

then update package cache. or add key and point repository to http://pub.thaiopensource.org/suriyan-repo

  • wget -q -O- http://pub.thaiopensource.org/suriyan-repo/suriyan-repo.key | sudo apt-key add -
  • apt-get update
